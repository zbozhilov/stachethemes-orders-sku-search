A WooCommerce extension that allows you to Search Orders by Product SKU in the Admin Panel.

== Description ==
This extension adds an additional "Product SKU" search filter to your WooCommerce Orders page, enabling you to search for orders by product SKU.
Supports HPOS and Legacy Modes.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Click **Upload Plugin** and upload the plugin archive.
3. Install and activate the plugin.

= Manual installation =

1. Upload the entire `stachethemes-orders-sku-search` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Stachethemes Orders SKU Search plugin.
